简单解决“抱歉，您所指定的页面不存在”

发表于 2018-09-12 | 分类于 nas

刚刚出现，黑群晖3617的，无论是 HTTP 还是 HTTPS 都是一样的结果，无法访问。
搜索了下，遇到这个问题的人很多，很多人都说要重装，硬盘数据这么多，很麻烦啊。还有的说是群晖封的，还好，被我找到解决办法了。

老外的原帖：https://xpenology.com/forum/topic/12406-how-to-fix-sorry-that-page-doesnt-exist/

找一台正常的群晖主机，复制 lib 目录下其中的两个文件：libsynopkg.so.1 和 libsynoshare.so.6。
或者，官网直接下载 ds3617xs 最新版的安装包，直接用7zip解压，在目录 hda1.tgz\hda1\usr\lib 里复制出来；

用 winscp 连接黑群晖主机，通过 sftp 或者 scp 连接，我的黑群晖好像只能用 scp 连接，将上述两个文件复制到 tmp 文件夹里；

用 putty ssh 的方式连接主机，通过命令行将 tmp 文件夹里的两个文件移动到 lib 目录下；
